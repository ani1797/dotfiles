# Module Documentation

Each module resides in its own directory under the root of the repository. The directories contain the dotfiles that will be stowed into your home or work directories.

## bash
Contains Bash configuration files such as `.bashrc`, `.bash_profile`, and any related scripts.

## vim
Holds Vim configuration (`.vimrc`) and optional plugin directories.

## zsh
Includes Zsh configuration files like `.zshrc` and any related plugins.
